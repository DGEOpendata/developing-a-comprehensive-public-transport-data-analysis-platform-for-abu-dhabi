# Comprehensive Public Transport Data Analysis Platform for Abu Dhabi

## Overview
This repository provides a Python-based implementation of a public transport data analysis platform for Abu Dhabi. The platform leverages the public transport dataset of 2026 to provide insights into passenger volumes, travel trends, and service punctuality.

### Key Features:
- Analysis of peak and off-peak travel trends.
- Visualization of passenger volume patterns.
- Calculation of service punctuality metrics.

## Prerequisites
1. Python 3.8 or higher installed on your machine.
2. Required Python libraries: pandas, matplotlib.
3. The dataset file `Abu_Dhabi_Public_Transport_Usage_2026.csv` downloaded from the Abu Dhabi Open Data Platform.

## Installation
1. Clone this repository:
   bash
   git clone https://github.com/your-repo/transport-analysis-platform.git
   cd transport-analysis-platform
   

2. Install the required Python libraries:
   bash
   pip install pandas matplotlib
   

3. Place the `Abu_Dhabi_Public_Transport_Usage_2026.csv` file in the repository folder.

## Usage
Run the script by executing the following command:
bash
python analyze_transport_data.py


## Example Output
1. A bar graph will be generated showing passenger volumes during peak and off-peak hours.
2. The average punctuality percentage will be printed in the terminal.
